## 🏁 QUICK START

Get the engine running in under 60 seconds.

### 1. Initialize The Core
Clone the repository and install the dependencies.
```bash
git clone https://github.com/bharathkumar000/madumaga.git
cd madumaga
npm install
```

### 2. Configure The Matrix
Duplicate the `.env.example` file to create your local `.env` configuration.
```bash
cp .env.example .env
```
Populate it with your Supabase credentials:
```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 3. Ignite
Start the Vite development server.
```bash
npm run dev
```
Navigate to `http://localhost:5173` to experience MADUMAGA.

## 🏗️ ARCHITECTURE

MADUMAGA utilizes a component-driven architecture prioritizing reusability, strict separation of concerns, and immediate visual feedback. State is managed locally for optimistic updates and continuously synchronized with Supabase's real-time channels to ensure absolute data integrity.
